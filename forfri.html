Descripción del Proyecto

Forfri es una aplicación de ventas diseñada para facilitar el proceso de compra y venta de productos. Utiliza tecnologías como React Native para el desarrollo móvil y Firebase para la gestión de datos en tiempo real, ofreciendo una experiencia de usuario fluida y eficiente.

Tecnologías Utilizadas

  • React Native
  • Firebase

Características Clave

  • Interfaz de usuario amigable
  • Notificaciones en tiempo real
  • Gestión de inventario
